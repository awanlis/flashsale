# flashsale

A **Next.js** dashboard UI for a “Flash Sales Report”, with charts, tables, and theme support.

## Tech Stack
- Next.js \`16.0.8\`
- React \`19.2.1\`
- TypeScript
- Tailwind CSS v4
- D3 (charts)
- lucide-react (icons)
- next-themes (theme toggle)

## Requirements
- **Node.js >= 20.9.0** 
- npm 

## Setup
1. Install dependencies:
   ```
   npm install
   ```

2. Run the dev server:
   ```
   npm run dev
   ```

3. Open:
   http://localhost:3000

## Scripts
- **Start dev server:**
  ```
  npm run dev
  ```
- **Production build:**
  ```
  npm run build
  ```
- **Run production server:**
  ```
  npm run start
  ```
